📊 MIS Analyst Dashboard | Sales & Performance Analysis
📌 Project Overview

This project showcases an MIS (Management Information System) Dashboard designed to provide actionable insights into sales, operations, and business performance.

The dashboard enables management to:

Track KPIs in real-time
Monitor sales and order performance
Analyze operational efficiency
Support data-driven decision-making
